# Health Monitor – Mi Band 4 + Raspberry Pi

Projeto de monitoramento de saúde contínuo (24/7) usando
Mi Band 4, Raspberry Pi e Python, com foco em estabilidade,
confiabilidade e baixo custo.

## Objetivo

Criar um sistema capaz de:
- coletar batimentos cardíacos em tempo real
- detectar anomalias
- enviar alertas confiáveis
- funcionar sem cloud paga
- rodar 24/7 em ambiente doméstico

## Estrutura do projeto

- `core/` → scripts de monitoramento
- `tools/` → ferramentas de teste BLE
- `docs/` → documentação técnica
- `logs/` → exemplos de execução

## Aviso importante

Este projeto envolve engenharia reversa e testes BLE.
Use apenas em dispositivos de sua propriedade.
